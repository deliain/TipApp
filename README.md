# TipApp

TipApp is a simple Android application that helps users quickly calculate tips and split bills when dining with others.

## Features

- Enter the total **bill amount**
- Select the **tip percentage** using a slider
- Automatically **calculate the tip amount**
- **Split the bill** between multiple people
- Clean and simple user interface

## How It Works

1. Enter the total bill amount.
2. Use the slider to choose the desired tip percentage.
3. The application calculates the tip automatically.
4. Enter the number of people to divide the bill.
5. The app displays the **amount each person needs to pay**.

## Example

Bill Amount: €100  
Tip: 10%  
Total Bill: €110  

If 2 people share the bill:

Each person pays: €55

## Tech Stack

- Kotlin
- Jetpack Compose
- Android SDK
- Android Studio

## Installation

1. Clone the repository: https://github.com/deliain/TipApp.git
2. Open the project in **Android Studio**.
3. Build and run the application on an emulator or Android device.
